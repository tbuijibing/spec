## 完整示范模块结构（SPEC-101）

```text
SPEC-101-用户画像模块/
├── README.md                       ← 模块导航
├── SPEC-101-1-需求文档.md           ← 用户故事 + 验收标准
├── SPEC-101-2-技术设计文档.md       ← 架构 + API + 关键实现

├── SPEC-101-4-任务文档.md           ← 阶段任务 + DoD 清单
├── SPEC-101-5-验收文档.md           ← 全量测试规范
├── SPEC-101-6-数据库文档.md         ← DDL + 索引
├── SPEC-101-7-快速排查手册.md       ← 常见错误 + 排查步骤
├── SPEC-101-8-可追溯性矩阵.md       ← 需求↔API↔代码↔测试
├── SPEC-101-9-验收报告.md           ← 终版验收结论
├── SPEC-101.pen                    ← 最新 Pencil 设计文件
├── assets/                         ← 最新设计图
└── versions/
    ├── v1.0/                       ← 初版归档
    │   ├── 变更记录.md
    │   ├── assets/
    │   └── test-report/

        ├── 变更记录.md
        ├── assets/
        └── test-report/

            ├── v1.0.1.01/          ← 第1轮（发现 bug）
            └── v1.0.1.02/          ← 第2轮（修复后通过）
```
