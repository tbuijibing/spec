## 开发铁律

1. **严格串行**：任意一个模块必须完成代码实现 + 浏览器全流程验收通过 + 截图存档 + 验收报告填写，才允许开始下一个。
2. **禁止并行**：同一时间只允许一个模块处于「进行中」状态。
3. **验收不通过 = 未完成**：代码写完 ≠ 验收通过。
