# SPEC-101 用户画像模块 v1.0 变更说明（初版）

> 版本：v1.0 | 发布日期：2026-03-29 | 状态：✅ 已验收

---

## 一、版本目标

初始实现用户画像模块：用户画像 CRUD、标签管理、ONE ID 合并、批量导入导出、画像完整度计算。

---

## 二、需求变更

初始版本，无前版需求变更。

---

## 三、新增功能

| 功能 | 类型 | 说明 |
|------|------|------|
| 用户画像 CRUD | 新增 | 基础信息 + 6 个 JSON 扩展字段 |
| 标签管理 | 新增 | 添加/删除/搜索，JSON 数组存储，自动去重 |
| 画像完整度计算 | 新增 | 0-100 分，按字段权重自动计算 |
| ONE ID 合并 | 新增 | 跨渠道账号合并，事务保证 |
| 批量导入/导出 | 新增 | Excel，最大 1000 条/次 |
| 行为时间线 | 新增 | 展示用户操作历史（只读） |
