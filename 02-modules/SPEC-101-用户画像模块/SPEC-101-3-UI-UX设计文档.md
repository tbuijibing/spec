## 3. 新增/编辑 Dialog（UIX-USR-FORM-001）

宽度 60%，6 个 Tab 页签：

| Tab | 包含字段 |
|-----|----------|
| 基础信息 | real_name, gender, birthday, constellation, profession, income_range |
| 扩展信息 | family_members（动态列表）, anniversaries（日期列表） |
| 地址信息 | addresses（收货地址列表，支持多条）|
| 社交账号 | social_accounts（渠道 + 账号 ID 对） |
| 偏好设置 | preferences（JSON 键值对编辑器） |
| 完整度 | 只读，展示各维度得分 |

**表单验证：**
- `real_name`：必填，最长 50 字符
- `birthday`：不能晚于当前日期
- `income_range`：数字或选项

## 5. ONE ID 合并 Dialog（UIX-USR-MERGE-001）

```text
[ONE ID 合并] ← Dialog 标题
─────────────────────────────
  主账号：[搜索/选择用户]     → 保留账号
  被合并账号：[搜索/选择用户] → 合并后标记为 status=2
  
  ⚠️ 操作不可撤销，合并后被合并账号渠道绑定全量迁移至主账号
─────────────────────────────
  [取消]              [确认合并]
```
