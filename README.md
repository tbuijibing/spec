## 01-roadmap — 计划层

> **路线图 + 总体进度**，管理 17 个模块的开发顺序、批次、里程碑、DoD 状态。

| 文档 | 说明 |
|------|------|
| `路线图.md` | 当前最新全量路线图（17 模块，4 批次） |
| `进度追踪.md` | 实时总体进度（模块级 + DoD 级） |
| `versions/vX.Y/路线图.md` | 历史版本路线图归档 |

### 开发铁律

1. **严格串行**：一个模块完成代码 + 浏览器验收 + 截图存档 + 验收报告，才允许开始下一个。
2. **禁止并行**：同一时间只允许一个模块处于「进行中」。
3. **验收不通过 = 未完成**：代码写完 ≠ 验收通过。

## AI Skill — 一键操作

> Skill 文件位于 `.claude/skills/spec/`，在支持 Skill 的 AI 助手（Claude Code / Windsurf / Cursor 等）中直接使用 slash command。

| 命令 | 用途 | 触发场景 |
|------|------|----------|
| `/spec-new` | 新建模块 — 生成 9 类文档骨架 + 目录结构 | 开始一个新 SPEC 模块 |
| `/spec-audit` | 查漏补缺 — 扫描缺失文档/目录并自动修复 | 检查文档完整性 |
| `/spec-accept` | 验收归档 — DoD 核查 + 版本归档 + 更新进度 | 模块实现完成后 |
